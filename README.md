Installation
============

1. Clone repository:

    git clone git://github.com/dblanchet/dotvim.git ~/.vim

2. Create symlinks:

    ln -s ~/.vim/vimrc ~/.vimrc
    ln -s ~/.vim/gvimrc ~/.gvimrc

3. Switch to the `~/.vim` directory, and fetch submodules:

    cd ~/.vim
    git submodule init
    git submodule update
